# RSI（相对强弱指标）

RSI（Relative Strength Index）是 Wells Wilder 在 1978 年提出的动量振荡指标，用于衡量价格变动的速度和幅度，判断是否处于超买或超卖状态。

## 计算公式

RSI = 100 — [100 / (1 + RS)]

RS（相对强弱比）= N 日内平均上涨幅度 / N 日内平均下跌幅度

### 简化的 Wilder 计算方法

1. 计算每日价格变动：当日收盘价 — 前一日收盘价
2. 分别累加 N 日的上涨幅度和下跌幅度（下跌取绝对值）
3. 平均上涨幅度 = 上涨总幅度 / N
4. 平均下跌幅度 = 下跌总幅度 / N
5. RS = 平均上涨幅度 / 平均下跌幅度
6. RSI = 100 — 100 / (1 + RS)

## 参数标准

- 默认周期：14 天
- 常见周期：9 天（短线）、14 天（标准）、21 天（长线）

## 使用方法

### 超买超卖
- RSI > 70 → 超买，可能回调
- RSI < 30 → 超卖，可能反弹
- 在强趋势中，超买超卖阈值应适当放宽（80/20）

### 背离信号
- 价格创新高但 RSI 未创新高 → 顶背离 → 卖出信号
- 价格创新低但 RSI 未创新低 → 底背离 → 买入信号

### RSI 形态
- RSI 本身的头肩顶/底、支撑/突破也有分析价值
- RSI 在 50 上方为强势区域，50 下方为弱势区域

## 局限性

- 在强单边趋势中，RSI 会长期处于超买/超卖区，提早卖出可能错过主升浪
- 背离信号可能出现多次才真正反转，不宜单独作为交易依据
- 不同周期参数得出的 RSI 值不同，需结合交易策略选择
