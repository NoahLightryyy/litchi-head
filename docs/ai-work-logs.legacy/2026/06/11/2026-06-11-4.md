**不采纳项说明**：
- **信息隔离**（Analyst只看数据，Researcher只看报告）：不适用于四组大师场景——大师需要全貌来做风格化判断。改为"简报分区输出"的轻量适配。
- **单 Bull/Bear 对抗**：你的四组辩论架构本身就是差异化优势，不改为单组对抗。

### 核心发现
1. TradingAgents 的"多轮对抗"本质上不复杂——靠 Prompt 约束（"你必须反驳对方"），不是靠复杂代码
2. 你的 MemoryStore 基础设施比 TradingAgents 的纯文本 markdown 更先进，差的是"Agent 自动接入"这一层
3. 风控辩论和你现有的辩论编排是同一套模式——可以复用 `debate/` 基础设施
4. DeepSeek 的 token 成本使多轮辩论经济可行（GPT-4o 的 1/10）
