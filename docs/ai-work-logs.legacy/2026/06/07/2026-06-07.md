## 本次工作

### 1. AgentResult：从 dataclass 改为 Pydantic BaseModel + Generic[T]

**涉及文件**：
- 🔧 `src/agents/base.py` — AgentResult 核心改造
- 🔧 `tests/test_agents_base.py` — 新增 2 个 Generic 测试

**内容**：

将 `AgentResult` 从 `@dataclass` 改为了 `pydantic.BaseModel` + `Generic[T]`：

**变更要点**：
1. 新增 `T = TypeVar("T")` 泛型类型变量
2. `data: dict` → `data: dict | T`，默认 `{}`（向后兼容）
3. `to_message()` 新增 `isinstance(self.data, BaseModel)` 判断 → 自动 `.model_dump()` 序列化
4. `TYPE_CHECKING` 条件导入 `AgentMessage`，消除循环导入风险

**用法**：
```python
# 向后兼容（现有代码不动）
result = AgentResult(data={"key": "val"})

# 新写法（类型化输出）
class NewsOutput(BaseModel):
    summary: str
    sentiment: str

result = AgentResult[NewsOutput](data=NewsOutput(...))
result.data.summary  # Pyright 可静态校验 ✅
```

**与 ADR-008 设计差异**：
- ADR-008 设计用 `@dataclass` + `Generic[T]`
- 实际实现用 `BaseModel` + `Generic[T]`（与 ADR-001 统一，自带 `model_dump()`）
