### 3. MasterAgent 输出结构化

**性质**：重构（LLM 调用从 `ainvoke` → `invoke_structured`）

**新增模型**：
```python
class InvestmentAnalysis(BaseModel):
    rating: str       # 看涨/看跌/中性/谨慎/观望
    score: int        # 1-100
    summary: str      # 一句话观点
    analysis: str     # 详细分析
    key_evidence: list[str]  # 关键证据列表
    risk_warning: str | None  # 风险提示
```

**变更点**：
- `run()` 使用 `invoke_structured(InvestmentAnalysis)` 替代 `ainvoke()`
- `data["answer"]` 向后兼容（`summary + analysis` 拼接）
- `data["analysis"]` 新增结构化字段（`model_dump()`）
- 置信度公式：`0.6 + 知识命中因子 + LLM评分加权`

**测试更新**：
- 24 个 MasterAgent 测试全部适配新接口
- 集成测试验证结构化输出字段

**涉及文件**：
- `src/agents/master_agent.py` — 新增 `InvestmentAnalysis` + 修改 `run()`
- `tests/test_agents_master.py` — 所有 mock 从 `ainvoke` → `invoke_structured`
