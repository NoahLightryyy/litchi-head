- 透明使用，采集器自动缓存

#### 采集器（`src/data/collector.py`）— 15 测试

`DataCollector` 封装 akshare 六大接口：

| 方法 | akshare 函数 | 缓存 TTL |

| `get_all_stocks()` | `stock_info_a_code_name()` | 1h |
| `get_realtime_quotes()` | `stock_zh_a_spot_em()` | 30s |
| `get_realtime_quote(code)` | 从全市场过滤 | 30s |
| `get_klines(code, period, ...)` | `stock_zh_a_hist()` | 5min |
| `get_news(code)` | `stock_news_em()` | 2min |
| `get_industry_boards()` | `stock_board_industry_name_em()` | 1h |
| `get_concept_boards()` | `stock_board_concept_name_em()` | 1h |

错误处理：全部异常捕获 + 日志记录 + 返回空列表（"尽力而为"原则）
