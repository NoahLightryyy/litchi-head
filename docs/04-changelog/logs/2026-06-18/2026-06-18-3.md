## 完成内容

### TD-032 FallbackSource 恢复主源（~10min ✅）

**问题**：`fallback.py:109` 一旦切到备用源，永不尝试恢复主源，系统无限期停留在降级路径。

**修复**：备用模式每次先尝试主源，成功则自动切回（`_call()` 重构）：
```python
if self._using_fallback[endpoint]:
    try:
        result = primary_fn()           # 先试主源
        self._using_fallback[endpoint] = False  # 成功则切回
        return result
    except Exception:
        pass                            # 失败继续用备用
    try:
        result = fallback_fn()          # 备用兜底
        return result
    except Exception:
        ...
```

**验证**：84 Provider tests passed, Pyright 零错误
