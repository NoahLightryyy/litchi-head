### 设计要点

BiasReport 从 `direction_distribution`（Bullish / Bearish / Neutral）计算三个指标：
- **总体偏斜度** `(Bullish - Bearish) / Total` (−1 ~ +1)
- **共识强度** `max(Bullish, Bearish, Neutral) / Total` (0~1)
- **共识类型** `Bullish | Bearish | Neutral | Divided`（>0.5 过半判定）

关键设计：**区分 Neutral 和 Divided**——偏斜度为 0 时，全体观望（Neutral）和深度分歧（Divided）是两种不同信号。
