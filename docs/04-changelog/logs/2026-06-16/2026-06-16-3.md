## 本次工作

### 1. 交易平台全链路调研

**调研范围**：
- 国际：TradingView（1亿用户、400+指标、Pine Script）、Robinhood（极简）、Webull（中间路线）
- 国内：东方财富（9.16分，一站式+股吧）、同花顺（9.16分，Level-2+问财AI）、雪球（8.66分，UGC+跟投）
- AI 趋势：eToro Tori + Grok、Quant（10万人排队）、同花顺问财、Webull MCP Server

**核心发现**：
1. 全球交易平台正在从「仪表盘」转向「对话式 AI + Agent」
2. litchi-head 的「多 Agent 辩论决策」踩中了 Agentic AI + 投资决策 最前沿方向
3. 散户需要的不只是数据，而是「自上而下的决策推理链」

### 2. 产品定位确认

**最终定位**：专业看盘型 +「自上而下决策漏斗」
```
宏观总览 → 板块排行 → 产业链分析 → AI 辩论决策
```

**与用户的 3 轮对话确认**：
1. 方向：专业看盘型（非 AI 决策优先，也非综合仪表盘）
2. 数据：基础行情 + 技术指标 + 资金流向 + AI 解读（全部都要）
3. AI 角色：嵌入个股页，不跳转独立页面
4. 筛选模式：自动初筛 + 可干预（任一层级可停）

### 3. 前端脚手架搭建

**技术选型**（用户确认直接上 React，放弃 Streamlit）：
- 前端：Next.js 16 + React 19 + TypeScript + Tailwind v4 + shadcn/ui
- K 线图：Lightweight Charts（TradingView 开源版，12KB）
- 辅助图：ECharts
- API 层：FastAPI 桥接（Python，不改造现有代码）
- 状态管理：TanStack Query（API 缓存）+ Zustand（全局状态）

**关键设计决策**：
- 三页路由：`/` → `/sector/[id]` → `/stock/[code]`
- 暗色主题：Bloomberg `#0D1117` × TradingView `#2962FF`
