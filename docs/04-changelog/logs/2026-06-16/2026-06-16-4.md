### 4. Batch 2: 前端三页 mock → 真实 API 接入

**后端 routes 数据形状对齐**（`backend/routers/market.py` + `stocks.py`）：
- IndexQuoteResp → MarketIndex（补 `change` 字段）
- SectorItemResp → SectorItem（补 `heat/top_stocks/rank`，用本地增强字典）
- SectorDetailResp → SectorDetail（补 `change_pct/fund_flow/heat/ai_analysis/stocks`）
- MacroBriefResp → MacroBrief（补 `generated_at/market_style/hot_topics`）
- `/api/stocks/{code}/quote` 补 `turnover_rate/fund_flow/market_cap` 字段 + `open_` → `open` 映射

**三页组件化重写**：

| 页面 | 旧方案 | 新方案 |
|:-----|:-------|:-------|
| `/` 宏观总览 | 内联 MOCK_INDICES/SECTORS/BRIEF + 内联子组件 | `useMarketIndices/useSectors/useMacroBrief` hooks + `MarketIndices/SectorRanking/MacroBrief` 组件 |
| `/sector/[id]` 产业链 | 内联 MOCK_SECTOR_DETAIL + 内联渲染 | `useSectorDetail` hook + `SectorHeader/ChainMap/ChainAnalysis/StockList` 组件 |
| `/stock/[code]` 个股 | 内联 MOCK_QUOTE + 内联 DebatePanel | `useStockQuote/useKline/useStockNews` hooks + `QuoteCard/KlineChart/DebatePanel/NewsFeed` 组件 |

**新增**：
- `providers.tsx` — QueryClientProvider 全局注入
- 搜索 autocomplete（`useStockSearch` → 实时下拉建议）
- 三态覆盖：每个页面都有 loading（骨架屏）/ error（错误卡片+返回按钮）/ data 状态
