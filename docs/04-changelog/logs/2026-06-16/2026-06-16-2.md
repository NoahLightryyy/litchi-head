**内容**：
实现 M4 动态权重，将 M3 TrustTracker 的信任度因子接入 aggregate 投票聚合流程：
1. `VoteSummary.trust_weight_factors` — 记录本次聚合使用的信任度因子
2. `DebateState.trust_weight_factors` — 状态传递字段
3. `DebateOrchestrator(enable_trust=True)` — 启用开关，接受 memory_store 时自动预加载因子
4. `aggregate_node` — 读取 trust_weight_factors，与 D3 weight_suggestions 相乘得到 combined_factor
5. 安全降级 — 无 memory_store / 无信任度记录 / 不可靠数据 → 因子=1.0

**权重叠加逻辑**：
```
combined_factor = D3_weight_suggestion × M4_trust_factor
weighted_score = Σ(score × confidence × combined_factor) / Σ(confidence × combined_factor)
```
