### 2. DataCollector 重构

`src/data/collector.py` 从"直调 akshare"改为"委托给 DataSource"，API 完全向后兼容：

```python
# 默认行为不变
collector = DataCollector()

# 切到 adata
collector = DataCollector(source=ADataSource())

# 主备自动切换（推荐）
source = FallbackSource(primary=ADataSource(), fallback=AKShareSource())
collector = DataCollector(source=source)
```
