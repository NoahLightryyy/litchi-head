> 本会话：深度讨论项目方向，确立"虚拟小投行"设计哲学，产出设计蓝图文档，将 7 项 DP 任务下放到各部门。

## 会议决议

### 核心理念确认

1. **小投行体系 > 超天才模型** — 不是做一个更好的交易机器人，而是模拟一个组织
2. **公式负责调整，镜子负责展示，人负责拍板** — 三权分立原则
3. **只留 DeepSeek** — 单 provider 策略，保留 `get_llm(provider=...)` 接口供未来扩展
4. **不做**多模型混搭 / RL 强化学习 / 自然语言反思自动注入 / 多用户社交 / Agent 拟人化制度

### D1 辩论方向修正

从"反驳"（adversarial）改为**"同侪审阅"**（赞同 + 补充 + 异议三段式），防止辩论过拟合和观点极端化。

### 调研确认

| 主题 | 结论 |
|:-----|:------|
| 投行机制映射 | 能套的=机制/原则（信息隔离、偏斜公示、复盘追责），不能套的=制度/组织（薪酬、晋升、合规文书） |
| 行业竞品 | TradingAgents(69k⭐)、FundaPod、BestAITrader 与你思路高度一致，方向被验证 |
| 反馈机制 | 自然语言反思已被 ATLAS 论文证伪，走公式调整+镜子+人决策 |
