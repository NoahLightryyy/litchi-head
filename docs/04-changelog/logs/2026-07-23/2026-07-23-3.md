## 本次工作

### FD-002：估值比率模型（17 新增测试 ✅）

**数据模型 `src/data/models.py`**：
- 新增 `ValuationMetrics(BaseModel)` — 5 字段：
  - `pe` — 市盈率 = 股价 / EPS（亏损公司为 0.0）
  - `pb` — 市净率 = 股价 / 每股净资产
  - `ps` — 市销率 = 总市值 / 主营业务收入
  - `market_cap` — 总市值(元)（从 StockQuote 获取）
- `StockQuote` 新增 `market_cap` 字段（默认 0.0，向后兼容）

**设计决策：纯计算模型，不新增 DataSource 协议方法**
- PE/PB/PS 是派生指标（股价 × 财务数据），不是原始数据
- `DataCollector.get_valuation(code)` 组合 `get_financials()` + `get_realtime_quote()` 计算
- 缓存 TTL = 5 分钟（随股价变化）

**AKShareSource 扩展**：
- `_row_to_quote()` 新增 `row.get("总市值", 0.0)` 映射
