# 数据采集模块 — 架构决策

## ADR-003: akshare 作为首要 A 股数据源

| 字段 | 值 |
|------|-----|
| **日期** | 2026-06-03 |
| **状态** | ✅ 已采纳（MVP 阶段） |
| **影响范围** | `src/data/` |

**上下文**：
A 股数据源选择，需要免费、API 稳定、覆盖全面。

**决策**：
MVP 阶段使用 `akshare` 作为唯一数据源。

**限制**（技术债务）：
- 非官方数据源，宕机无 SLA 保障
- 请求频率受限
- 不支持 Level2 行情

**未来方案**：
Phase 2 考虑接入同花顺 iFinD / 聚宽数据。
