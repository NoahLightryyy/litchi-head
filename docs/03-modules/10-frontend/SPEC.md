# 前端规格

## 1. 产品定位

专业散户一站式投资决策工具。以「自上而下决策漏斗」为核心交互模型：
宏观抓方向 → 板块找赛道 → 产业链定标的 → AI 辩论助决策。

## 2. 页面规格

### 2.1 Page 1: 宏观总览 `/`

**功能点**：
1. 三大指数实时行情（上证/深证/创业板）
2. 板块排行榜（默认按资金流向排序，可切换维度）
3. AI 宏观简报（LLM 自动生成当日的市场风格、热点、风险提示）
4. 关键词搜索（快速定位板块或个股，支持 autocomplete）
5. 热点快讯流（滚动）
6. 点击板块 → 跳转到 `/sector/[id]`

**数据源**：`backend/api/market/overview` + `market/sectors`

### 2.2 Page 2: 产业链分析 `/sector/[id]`

**功能点**：
1. 板块头部信息（涨跌幅、主力净流入、热度标记）
2. 产业链地图（上游 → 中游 → 下游，关键节点标记 ⭐）
3. 板块个股列表（可排序：AI 评级/资金流向/涨跌幅/市值等）
4. AI 产业链分析（自动分析产业链关键环节与不可替代企业）
5. 点击个股 → 跳转到 `/stock/[code]`

**数据源**：`backend/api/market/sector/{id}` + 本地产业链知识库

### 2.3 Page 3: 个股决策 `/stock/[code]`

**功能点**：
1. 实时行情卡片（价格、涨跌幅、成交量、换手率、主力净流入）
2. K 线图全宽主区域（Lightweight Charts）
   - 周期切换：日/周/月
   - 可叠加指标：均线(MA)、RSI、MACD、布林带
3. Tab 切换面板：
   - **技术分析**：指标详情 + 技术信号
   - **资金流向**：主力净流入趋势、大单追踪
   - **AI 辩论**：触发辩论 → 展示大师分析 + 共识评级
   - **信任度**：大师历史准确率 + 校准曲线
4. 关联新闻流
5. 「重新辩论」按钮

**数据源**：`backend/api/stocks/{code}` + `/kline` + `/news` + `/debate`

## 3. 路由规则

| 路径 | 页面 | 参数 | 说明 |
|:-----|:-----|:-----|:-----|
| `/` | 宏观总览 | — | 默认首页 |
| `/sector/[id]` | 产业链分析 | `id` = 板块代码 | 支持 `?sort=fund_flow` 排序参数 |
| `/stock/[code]` | 个股决策 | `code` = 股票代码 | 支持 `?tab=debate` 默认 Tab |

## 4. 状态管理

| 状态 | 存储 | 存活期 | 用途 |
|:-----|:-----|:------:|:-----|
| API 缓存数据 | TanStack Query cache | 5-30 分钟 | 行情、板块、K 线等非关键实时数据 |
| WebSocket 行情 | Zustand store | 会话 | 实时价格推送 |
| 导航历史 | Zustand store | 会话 | 面包屑、最近浏览 |
| 辩论结果 | TanStack Query cache | 会话 | AI 辩论结果 |
| 用户偏好 | localStorage | 持久 | 默认 Tab、排序偏好 |

## 5. 加载策略

| 场景 | 表现 | 技术 |
|:-----|:-----|:-----|
| 页面初次加载 | 骨架屏(Skeleton) | `loading.tsx` 或 Suspense |
| API 请求中 | 局部骨架/Spinner | TanStack Query `isLoading` |
| API 请求失败 | 错误卡片 + 重试按钮 | TanStack Query `onError` |
| K 线图加载 | 占位图 | Lightweight Charts loading API |
| 辩论进行中 | 进度条 + 阶段指示 | 轮询 `/api/debate/status/{id}` |

## 6. 错误处理

| 错误类型 | HTTP 状态 | 前端表现 |
|:---------|:---------:|:---------|
| akshare 无网络 | 503 | 显示「数据源不可用」+ 使用缓存 |
| 股票代码不存在 | 404 | 显示「未找到该股票」+ 搜索提示 |
| 辩论超时 | 504 | 显示「分析超时，请重试」+ 按钮 |
| LLM 调用失败 | 502 | 显示「AI 服务暂时不可用」+ 缓存常用分析 |

## 7. 测试策略

| 层级 | 工具 | 覆盖范围 |
|:-----|:-----|:---------|
| 组件测试 | Vitest + React Testing Library | 组件渲染、交互、错误态 |
| API 模拟 | MSW (Mock Service Worker) | API 调用→响应转换 |
| E2E | Playwright | 三页漏斗全链路、搜索→导航→辩论 |
| 视觉回归 | Storybook + Chromatic | 组件视觉一致性 |
