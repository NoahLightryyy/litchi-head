# 辩论决策引擎 — 架构决策

## ADR-005: 四组辩论架构 + 大师投票

| 字段 | 值 |
|------|-----|
| **日期** | 2026-06-03 |
| **状态** | ✅ 已采纳 |
| **影响范围** | `src/debate/`, `src/orchestrator/` |

**上下文**：
投资决策需要多角度分析，避免单一模型的偏见。

**决策**：
采用"多大师分析 → 交叉审阅 → 独立评审 → 投票加权汇总"的四层架构。

**理由**：
1. 多种投资哲学覆盖不同市场环境
2. 交叉质疑机制增加决策鲁棒性
3. 独立评审层模拟"投委会"流程
4. 每层可独立替换/升级

**舍弃方案**：
| 方案 | 舍弃理由 |
|------|---------|
| 单模型直接输出 | 缺乏辩论的鲁棒性 |
| 纯投票机制 | 缺少观点交锋过程 |
