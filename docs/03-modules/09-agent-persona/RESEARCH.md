# 功能模块：Agent 人格与提示词工程 — 调研分析

> 本文档记录该模块在行业中的竞争格局、竞品分析、架构对比等调研信息。

## 所属战线

人格设计不是独立战线，而是辩论决策战线的**灵魂层**——大师 Agent "像不像"就取决于这个模块。

[查看战线完整分析](../../99-archive/KNOWLEDGE_BASE.md#-战线-1多-agent-辩论决策)

### 在这个战场中的角色

```
辩论决策战线中，人格设计决定了Agent"像个什么角色"：

TradingAgents：
  分析师 → "我是{基本面/技术面/舆情/新闻}分析师，只分析自己的领域"
  研究员 → "我是{看多/看空}研究员，必须坚持立场"
  交易员 → "我是交易员，基于分析做交易提案"
  风控   → "我是{激进/保守/中性}风控官，评估风险"

AI Hedge Fund：
  Buffett  → "我是沃伦·巴菲特，我的投资原则是寻..."
  Burry    → "我是迈克尔·伯里，我关注尾部风险和做空机会..."
  Taleb    → "我是纳西姆·塔勒布，黑天鹅事件不可避免..."

你（litchi-head）：
  大师们   → 各有名称但提示词风格是否统一？人格是否鲜明？
```

**核心认识**：大师 Agent 的人设不是一句"你是价值投资者"就完事的。需要把投资哲学编码为可执行的决策规则。

## 竞品分析

| 项目 | 人格设计方式 | 值得看的 |
|------|------------|---------|
| **AI Hedge Fund** | 14位大师各有具体投资规则，不是笼统的描述 | 每个大师有可执行的决策规则（如 Buffett：寻找护城河、管理层优秀、安全边际）|
| **TradingAgents** | 分层提示词——每层Agent只看到自己需要的信息 | 分析师不看辩论内容，交易员不看原始数据——信息隔离设计 |

## 架构对比分析

### AI Hedge Fund 的大师人格设计（值得看提示词工程）

```
Warren Buffett:
  - 寻找有护城河、管理层优秀的公司
  - 偏好长期持有，不考虑短期波动
  - 安全边际是核心购买决策要素
  - 不懂的行业不碰
  - 输出格式：{ticker, 分析, 建议, 置信度}

Cathie Wood:
  - 关注颠覆性创新（AI/基因/区块链/能源）
  - 容忍高波动，追求高增长
  - 看5-7年后的趋势而非当下
  - 重视营收增长率而非当前利润
  - 输出格式：{ticker, 分析, 建议, 置信度}

Ben Graham:
  - 寻找股价低于内在价值的公司
  - 用净流动资产（NCAV）筛选
  - 安全边际至少 30%
  - 分散投资降低风险
  - 输出格式：{ticker, 分析, 建议, 置信度}
```

关键点：**每个大师的人设有具体的、可执行的决策规则**，不只是"你是一个价值投资者"。

### TradingAgents 的分层提示词策略（信息隔离）

```
分析师层提示词：
  "你是{基本面/技术面/舆情/新闻}分析师。
   你的任务是分析{具体数据}，输出结构化报告。
   不要做交易决策，不要超出你的专业范围。"

研究员层提示词：
  "你是{看多/看空}研究员。
   你必须基于分析师报告形成{看多/看空}观点。
   阅读对方观点后必须反驳。
   你的输出将传递给交易员。"

交易员+风控层提示词：
  "你是交易员/风控Agent，你的任务是..."
```

关键点：**每一层 Agent 的提示词只告诉它这一层该做的事，不给全貌**：
- 分析师不知道辩论内容
- 交易员不直接看原始数据
- 研究员只基于分析师报告，不自己查数据

## 研究问题

### 人格设计
- [ ] 你的大师人设和 AI Hedge Fund 的"大师人格"有什么本质区别？
- [ ] 每个大师的决策规则是否足够具体？（不要"你是价值投资者"，要"你的安全边际要求至少20%"）
- [ ] 大师的"性格"如何影响决策？（保守大师是否天然倾向于中性？激进大师是否更容易满分买入？）

### 提示词工程
- [ ] 系统提示词 vs 用户提示词：你的 Agent 怎么区分"你是谁"和"当前任务"？
- [ ] 输出约束：Pydantic schema vs 自由输出 vs few-shot，哪种效果最好？
- [ ] DeepSeek（你的主力模型）和 Claude 的提示词写法有什么差异？
- [ ] 提示词版本管理：每次改提示词怎么追踪影响？

### 信息隔离
- [ ] TradingAgents 的做法——分析师只看数据、研究员只看分析、交易员只管交易——你的大师是否也需要做这层隔离？
- [ ] 如果你的大师既看数据又做决策，会不会产生"锚定效应"？（先看了某条新闻，后面分析时被影响了）

## 深挖方向建议

**优先级 1**：梳理你目前所有大师的提示词，统一模板格式。确保每个大师有具体、可执行的决策规则（而不是笼统的描述）。

**优先级 2**：研究 AI Hedge Fund 的大师人格提示词设计，和你的对比差异。特别是"Buffett 会怎么判断"这种人格化推理的提示词写法。

**优先级 3**：研究 TradingAgents 的分层提示词策略（分析师/研究员/交易员/风控各层信息隔离），评估是否适用于你的场景。
