## 模块定义

Alpha 因子挖掘、特征工程、AI 模型训练、因子有效性评估。

**职责边界**：
- ✅ Alpha 因子的生成与计算
- ✅ 因子有效性验证（IC / Rank IC / 回测）
- ✅ AI 模型训练（LSTM / Transformer / LightGBM 等）
- ✅ 自动因子挖掘（RD-Agent 风格）
- ❌ 不负责因子如何加权组合成交易决策（那是辩论模块的事）

## 架构（当前状态）

```
因子研究模块                    辩论决策模块
  ┌──────────┐                  ┌──────────┐
  │ Qlib:    │ → 因子信号 →    │ 大师分析  │
  │ 什么因子  │                  │ 时参考因子 │
  │ 有效     │                  │ 信号     │
  └──────────┘                  └──────────┘
                                       ↓
                                 最终交易决策
```

Qlib/RD-Agent 回答的是"什么因子有效"，你的系统回答的是"综合因子+新闻+行情该不该买"。两条线是**互补关系**，不是竞争关系。
