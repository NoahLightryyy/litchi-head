### 2. MasterAgent 实现（路径 A-3 核心）

**涉及文件**：
- ⭐ `src/agents/master_agent.py` — 全新通用大师 Agent（~130 行）
- ⭐ `tests/test_agents_master.py` — 24 个测试

**三种初始化方式**：
```python

MasterAgent(skill_id="buffett")               # 自动构建 SkillDisk
MasterAgent(skill_id="munger", disk=disk)     # 指定 disk
```

**工作流**：
1. 接收 question → 输入验证
2. KB.search() 语义检索（不再使用 knowledge_filter 作为 type 过滤，留待 Phase 1 文本级过滤）
3. skill.system_prompt + 知识上下文 → `llm_service.ainvoke()`

**降级策略**：知识未命中时纯 LLM 回答（confidence=0.6）
